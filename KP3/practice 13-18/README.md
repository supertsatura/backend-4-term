## Описание проекта

В рамках контрольной работы №3 разработано прогрессивное веб-приложение (PWA) для управления заметками с поддержкой:

- Офлайн-режима
- Установки на устройство
- App Shell архитектуры
- WebSocket (Socket.IO)
- Push-уведомлений
- Напоминаний с возможностью откладывания


## Практика 13-14: Service Worker + Web App Manifest

### Функционал
- Регистрация Service Worker для кэширования статических ресурсов
- Офлайн-доступ к приложению
- Манифест для установки PWA на устройство
- Иконки разных размеров (16x16 до 512x512)
- Сохранение заметок в localStorage

### Запуск
- bash
- cd pr13-14
- npx http-server -p 8080


## Практика 15-16: HTTPS + App Shell + WebSocket + Push


### Функционал
- HTTPS сервер с самоподписанным сертификатом
- App Shell архитектура (каркас + динамический контент)
- WebSocket (Socket.IO) для обмена данными между клиентами
- Push-уведомления через VAPID
- Кнопки включения/отключения уведомлений
- Страницы "Главная" и "О приложении"

# Запуск
- bash
- cd pr15-16
- npm install
- node generate-cert.js
- node server.js


## Практика 17: Детализация Push (Напоминания)


### Функционал
- Добавление заметок с указанием даты и времени напоминания
- Серверное планирование push-уведомлений (setTimeout)
- Push-уведомления с кнопкой "Отложить на 5 минут"
- Визуальное отображение отложенных напоминаний в списке
- Обновление времени напоминания в интерфейсе


### Запуск
- bash
- cd pr17
- npm install
- node generate-cert.js
- node server.js

## Выводы
В ходе выполнения контрольной работы №3 были освоены следующие темы:
1. Service Worker – кэширование ресурсов, офлайн-режим
2. Web App Manifest – установка PWA на устройство
3. HTTPS – настройка защищённого соединения
4. App Shell – архитектура для мгновенной загрузки
5. WebSocket (Socket.IO) – двусторонняя связь в реальном времени
6. Push-уведомления – отправка уведомлений даже при закрытом приложении
7. Детализация Push – напоминания с откладыванием

Все практические занятия выполнены в полном объёме, приложение работает стабильно в офлайн-режиме и корректно обрабатывает push-уведомления.
